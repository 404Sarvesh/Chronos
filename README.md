# Chronos

A sleek and minimalist timer web app with futuristic design and interactive features.

## About

Chronos Elite is a minimalist timer application with a sophisticated, space-inspired aesthetic. It combines functionality with visual appeal, featuring a glassmorphic UI, interactive particle animations, and customizable settings.

## Features

- **Intuitive Timer Control**: Set hours, minutes, and seconds with easy increment/decrement controls
- **Dual Display Modes**: 
  - Large, luminous timer display with glowing text effects
  - Compact control panel with adjustable time values
- **Interactive Particle System**: Mouse-responsive background particles that react to your cursor movement
- **Customizable Themes**: 
  - Adjustable background colors with automatic contrast detection
  - Glow intensity slider for personalized visual experience
- **Stealth Mode**: UI automatically hides during timer countdown for distraction-free timing
- **Fullscreen Support**: Dedicated fullscreen toggle for immersive experience
- **Glassmorphic Design**: Modern glass-effect UI with blur effects and smooth transitions
- **Responsive Layout**: Adapts perfectly to different screen sizes using Tailwind CSS
- **Premium Typography**: Uses Orbitron and JetBrains Mono fonts for a futuristic aesthetic

## How to Use

1. **Set Time**: Use the +/- buttons to adjust hours, minutes, and seconds
2. **Start Timer**: Click the "Start" button to begin the countdown
3. **Pause/Resume**: Click "Pause" to pause and "Resume" to continue
4. **Reset**: Use the reset button to return to your preset time
5. **Customize**: Click the settings icon (⚙️) to adjust glow intensity and background color
6. **Fullscreen**: Click the fullscreen icon to expand to full screen

## Design Highlights

- **Space-Inspired Aesthetic**: Dark background with glowing elements and particle animations
- **Glassmorphic UI**: Frosted glass effects with backdrop blur for modern appeal
- **Color Contrast Detection**: Automatically determines optimal text color based on background
- **Smooth Animations**: Sleek transitions and fade effects throughout the interface
- **Neon Glow Effects**: Text shadow and particle glow effects create a futuristic ambiance

## Technology Stack

- HTML5 Canvas for particle animations
- Tailwind CSS for responsive styling
- Vanilla JavaScript for timer logic
- Custom CSS animations and transitions
